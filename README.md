# Habit-Matrix

## Description

This is a web application that allows users to track their habits and manage their tasks effectively.

## Installation

1. Clone the repository:git clone https://github.com/your-username/habit-tracker.git
   
2. Install the dependencies: cd habit-tracker
npm install

3. Start the development server:

npm start

4. Open the app in your browser at [http://localhost:3000](http://localhost:3000).

## Usage

- Create a new task by entering the task name and description.
- Click the "Add The Task" button to add the task to the list.
- View the list of tasks and their details on the home page.
- Navigate to the "Details" page to see weekly updates and track task completion.

## Features

- Add tasks with names and descriptions.
- Delete tasks from the list.
- Track task completion on a weekly basis.
